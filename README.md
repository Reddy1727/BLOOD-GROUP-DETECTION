# Blood Group Detection Using Fingerprint

## Overview

Blood Group Detection Using Fingerprint is a Deep Learning based system that predicts a person's blood group from fingerprint images. The project utilizes image processing and convolutional neural network (CNN) architectures to classify fingerprint patterns into different blood groups.

## Features

* Blood group prediction using fingerprint images.
* Deep Learning-based classification.
* User-friendly web interface developed using Python.
* Performance analysis with accuracy and loss graphs.
* Comparison of multiple CNN models such as AlexNet, LeNet, ResNet34, and VGG16.

## Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* NumPy
* Pandas
* Flask
* Jupyter Notebook

## Project Structure

* `app.py` – Main application file.
* `train_model.py` – Model training script.
* `create_model.py` – Deep learning model creation.
* `dataset/` – Fingerprint image dataset.
* `sample dataset/` – Sample images for testing.
* `graphs/` – Training and validation performance graphs.
* `performance metrix/` – Accuracy and performance comparison results.
* `test/` – Testing files and notebooks.

## Installation

1. Clone the repository:
   git clone https://github.com/Reddy1727/BLOOD-GROUP-DETECTION.git

2. Navigate to the project folder:
   cd BLOOD-GROUP-DETECTION

3. Install dependencies:
   pip install -r requirements.txt

4. Run the application:
   python app.py

## Dataset

The dataset consists of fingerprint images categorized according to blood groups. Images are preprocessed and used for training and testing deep learning models.
